# UploadFile 组件 (目前只支持单个文件)

## Usage

```jsx
import {UploadFile, FormItem} from '@mced/components';

return (
  <FormItem {...formitemProps}>
    <UploadFile {...uploadProps} />
  </FormItem>
);
```

## Options

| 参数       | 说明         | 类型   | 默认值 |
| ---------- | ------------ | ------ | ------ |
| uploadText | 上传按钮文本 | string | 上传   |
